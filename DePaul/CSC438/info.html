In this class you will learn the following:
Concepts
Push vs Pull frameworks, the Model-View-Controller design, how to deal with concurrency, how to maintain state, layers of data persistance (session, database, cache), basics of relational databases, unit testing and functional testing, how to build a simple parser, how to build a complete a complete web application, common security issues and how to avoid them, scalability issues.
Python Commands
int/long, tuple, float, str, unicode, dict, for, while, if..elif..esle, try..except, def, class, lambda, pass, del
web2py objects
request, response, session, cache, T, DAL, Table, Field, Query, Set, Rows, Row
web2py helpers
A, B, I, DIV, SPAN, UL, LI, ... MENU, BEAUTIFY
web2py template language
{{extend ...}} {{include}} {{block ...}} {{end}} {{for ...}} {{if ...}}
